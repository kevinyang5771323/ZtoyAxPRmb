# Python图书管理系统 python591

## 项目概述

本项目是一个基于Python的个人书籍管理系统，具备图形化界面，旨在帮助用户有效管理个人书籍信息。

## 技术栈

- Python
- 图形化界面库（如Tkinter）

## 功能模块

### 1.1 系统基本功能

- **录入书籍信息**
  - 用户按一定格式输入书籍信息。
  - 系统确保书籍记录的唯一性，避免重复序号。

- **保存书籍信息**
  - 将书籍信息保存至文件。
  - 提示用户保存结果。

- **浏览书籍信息**
  - 显示所有书籍信息。
  - 支持分页显示。

- **查询书籍信息**
  - 支持按照书名、作者、出版社、类别进行查询。

- **删除书籍信息**
  - 选择书籍，确认删除操作。
  - 删除操作后更新文件。

- **修改书籍信息**
  - 选择书籍，进行信息修改。
  - 修改操作后更新文件。

- **按类别统计书籍数量**
  - 用户选择类别，系统统计并显示详细信息。

- **按书名排序书籍**
  - 支持书名升序或降序显示。
  - 同名书籍按序号排序。

- **退出系统**
  - 用户确认退出。

### 1.2 程序设计要求

- **图形用户界面**
  - 界面友好，操作简便。

- **逻辑独立性**
  - 功能定义在独立的函数中。

- **合理的提示机制**
  - 对于修改、删除等操作，系统提示确认。

- **程序鲁棒性**
  - 对不合法数据有错误处理机制。

- **数据实时更新**
  - 所有的数据变更都实时更新至文件中。

## 系统角色

- 系统用户：使用系统进行书籍管理。
